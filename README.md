# Student Enrollment Form - JPDB

## Description

This project is a Student Enrollment Form developed using HTML, Bootstrap, JavaScript, jQuery and JsonPowerDB (JPDB).

The application stores student records in the **STUDENT-TABLE** relation of **SCHOOL-DB** database.

Fields used:

* Roll-No (Primary Key)
* Full-Name
* Class
* Birth-Date
* Address
* Enrollment-Date

The application validates the Roll Number before allowing data entry.

### Functionalities

* Save new student records
* Update existing student records
* Reset form
* Primary key validation
* Automatic retrieval of existing records
* Form field validation

## Benefits of using JsonPowerDB

* Serverless database
* High performance
* Schema-free database
* Easy REST API integration
* Fast development cycle
* Simple and lightweight
* Low learning curve
* Real-time data processing

## Scope of Functionalities

1. Check whether Roll-No exists in the database.
2. Enable Save button for new records.
3. Fetch existing records automatically.
4. Enable Update button for existing records.
5. Validate all fields before database operations.
6. Store and update records in JsonPowerDB.

## Technology Stack

* HTML5
* Bootstrap 3.4.1
* JavaScript
* jQuery
* JsonPowerDB

## Database Details

Database Name: SCHOOL-DB

Relation Name: STUDENT-TABLE

Primary Key: Roll-No

## Description

This project is a Student Enrollment Form developed using HTML, Bootstrap, JavaScript, jQuery, and JsonPowerDB (JPDB).

The application stores student records in the STUDENT-TABLE relation of the SCHOOL-DB database.

### Fields Used

* Roll-No (Primary Key)
* Full-Name
* Class
* Birth-Date
* Address
* Enrollment-Date

The application validates the Roll Number before allowing data entry.

## Scope of Functionalities

* Add new student records
* Update existing student records
* Reset form data
* Validate all input fields
* Search records using Roll Number
* Store data in JsonPowerDB
* Retrieve existing records automatically

---

## Examples of Use

### New Student Entry

1. Enter Roll Number.
2. If the Roll Number does not exist, enter student details.
3. Click Save.
4. Record is stored in JsonPowerDB.

### Existing Student Update

1. Enter Roll Number.
2. Existing data is fetched automatically.
3. Modify required fields.
4. Click Update.
5. Record is updated successfully.
